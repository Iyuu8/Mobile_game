# Medieval Fantasy RPG

A 2D pixel art RPG mobile game built in Kotlin for Android.

## Features
- 5 playable classes: Knight, Bandit, Necromancer, Wizard, Shadow King
- Fantasy enemies: Goblins, Orcs, Trolls, Dragons
- Main story campaign (~30 minutes)
- Side quests and exploration
- Touch controls with virtual joystick
